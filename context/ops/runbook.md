## 장애 대응 SOP

### 사이트 접속 불가
1. https://status.vercel.com/ 확인 → Vercel 장애면 대기 + 트위터·블로그 모니터
2. 도메인 DNS 확인 (`dig www.ghpc.or.kr`) → Cloudflare 대시보드
3. Vercel Deployments → 최근 배포 실패 시 이전 배포로 Promote
4. 30분 내 복구 불가 → 미디어팀에 SNS·유튜브 공지 요청

### 폼 제출 실패
1. Sentry 에러 확인
2. Supabase 대시보드 → RLS 정책·테이블 상태 확인
3. service_role 키 만료·키 회전 이슈 → 키 재발급
4. 임시 우회: 폼 일시 비활성화 + 안내 메시지

### 라이브 영상 안 보임
1. YouTube 채널 직접 확인 — 송출 중인가
2. YouTube API 한도 초과 (`YOUTUBE_API_KEY` quota) → 시간표 폴백
3. video ID 비공개 처리됨 → 다시보기로 표시 전환

### Supabase 사용량 초과
1. 알림 받자마자 Supabase Dashboard 확인
2. 무거운 쿼리 식별 → 인덱스 추가 또는 페이지네이션
3. Storage 초과 → 오래된 콘텐츠 아카이브 또는 Pro 플랜 전환

### 데이터 실수 수정·삭제
1. **즉시 작업 중단** (추가 변경 금지)
2. 마지막 백업 시점 확인 (`context/ops/backup.md`)
3. Pro 플랜이면 PITR 복원
4. 무료 플랜이면 수동 스냅샷 + 직접 수정
