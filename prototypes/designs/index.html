메인 시안 4종

A안 콘텐츠 라이브러리형 리고니어 스타일

다크 네이비 에디토리얼 — 영상 없는 타이포 헤로, 스몰캡스 라벨·얇은 룰, 설교/칼럼/도서 콘텐츠 밀집 그리드, 사이트맵 푸터.

  • 말씀·콘텐츠의 권위감을 가장 강하게 표현
  • 설교/도서/칼럼 등 라이브러리 확장에 유리
  • 지적·신학적 톤 — 진중하고 차분한 분위기
A안 시안 열기 →
B안 모던 환영형 그레이스마린 스타일

풀블리드 사진 헤로 + 넓은 여백 — "처음 오시나요" 방문 안내 중심, 사진↔텍스트 교차 스토리텔링, 세이지 그린·크림 톤의 에어리한 분위기.

  • 새신자·방문자 친화도 가장 높음
  • 여백·사진 중심 — 따뜻하고 부담 없는 인상
  • 3040 세대 친화적 모던 톤
B안 시안 열기 →
C안 포털형 사랑의교회 스타일

슬라이드 헤로 + 퀵메뉴 그리드 + 공지/주보 게시판 — 정보 밀도 가장 높음. 오션 블루·딥 네이비 톤의 전통적 한국교회 포털 구조.

  • 기존 사용자(50대 이상)에게 익숙한 구조
  • 한 화면에서 모든 메뉴 즉시 진입 가능
  • 공지·주보·게시판 운영에 최적화
C안 시안 열기 →
D안 공동체 중심형 경향교회 자체

파치먼트 + 브랜드 네이비·크림슨 — A·B·C의 장점을 절충한 우리교회만의 톤. 따뜻한 공동체 사진·말씀 중심 카피·모던 한국교회 레이아웃.

  • 경향교회 정체성·전통과 가장 잘 맞는 톤
  • 새신자·기존 성도 양쪽에 무리 없는 균형
  • 실제 운영 단계에서 가장 확장성 높음
D안 시안 열기 →
공통 사항: 네 시안은 레퍼런스 사이트를 기반으로 레이아웃·컬러·분위기를 전면 차별화했습니다 — A: 리고니어(다크 네이비+크림슨, 콘텐츠 라이브러리) / B: 그레이스마린(크림+세이지, 풀블리드 사진) / C: 사랑의교회(딥 네이비+오션, 포털 그리드) / D: 경향교회 자체(파치먼트+네이비·크림슨, 공동체 중심). Pretendard 폰트·GNB 메뉴 구조는 공통이며, placeholder 텍스트·이미지는 실제 콘텐츠로 교체 예정입니다.
