# 원본 도면 (길찾기 SVG 변환 소스)

여기에 받은 도면 파일을 아래 **파일명 규칙대로** 넣어줘.
이 파일명을 기준으로 SVG 변환 → 길찾기 데이터에 매핑함.

## 파일명 규칙

| 층 | 파일명 | 상태 |
|---|---|---|
| 지상 1층 | `F1.*` | ✅ 수령 |
| 지상 2층 | `F2.*` | ✅ 수령 |
| 지하 1층 | `B1.*` | ✅ 수령 |
| 지하 2층 | `B2.*` | ✅ 수령 |
| 지하 3층 | `B3.*` | ✅ 수령 |
| 지하 1층 주차장 | `B1-parking.*` | ⏳ 미수령 |
| 지하 2층 주차장 | `B2-parking.*` | ⏳ 미수령 |

- 확장자는 `.ai` / `.png` / `.jpg` / `.pdf` 아무거나 OK (원본 그대로).
- 한 층에 여러 장이면 `B1-1.png`, `B1-2.png` 식으로 뒤에 번호.

## 주의

- `.ai` 같은 대용량 원본은 git에 커밋하면 레포가 무거워짐 → 커밋 여부는 이레가 판단
  (참고용 로컬 보관만 할 거면 `.gitignore`에 `plans/*.ai` 추가 권장).
- 주차장 도면은 나중에 받으면 같은 규칙으로 추가만 하면 됨 (기존 작업 영향 없음).
