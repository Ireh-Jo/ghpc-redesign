# 02. Information Architecture (IA)

5개 페이지의 메뉴 구조와 페이지별 섹션 우선순위. 기획서(PPT) 기준.

## 전역 GNB (5개 + 새가족 CTA)

```
[로고] 교회소개 | 예배와 교육 | 목양과 사역 | 교회 활동 | 새가족 [생방송 보기 CTA]
```

- **새가족** 메뉴는 별도 강조(다른 색 또는 우측 정렬) — 새신자 진입 최우선
- **생방송 보기** CTA는 헤더 우측 고정 — 주일/수요예배 시간엔 빨간 점 활성
- 데스크탑: 드롭다운으로 2뎁스 노출
- 모바일: 햄버거 → 풀스크린 메뉴

## 페이지별 IA

### 1. 메인 (`01-main.html`)

스크롤 우선순위 (위 → 아래):

1. **헤로 영상** — 교회전경 + 햇빛, 위에 짧은 환영 카피 + 주일예배 시간
2. **이번 주일 예배 안내** — 시간/장소/설교제목/말씀본문 카드
3. **설교 다시보기** — 최근 3개 영상 썸네일
4. **생방송 안내** — 다음 생방송 일정 + 진입 버튼
5. **Go & Grow 캠페인 배너** (팝업 또는 인라인) — 한해 캠페인
6. **새가족 환영** — 새가족 등록 CTA + 첫 방문 안내
7. **오시는 길 + 푸터**

### 2. 교회 소개 (`02-intro.html`)

1. **헤로** — "경향교회 50주년" 타이틀 + 50주년 영상 큰 임베드
2. **인사말** — 신승욱 담임목사 (텍스트 + 이미지 placeholder)
3. **비전 / 신학 노선** — 카드 3개 또는 4개
4. **역사** — 타임라인 형식 (1973 설립 → 2023 50주년)
5. **섬기는 사람들** — 담임/원로/은퇴 목사·장로·교역자 (그리드)
6. **교회 정보** — 위치(지도), 시설, 전화번호, 사역일정
7. **Q&A (AI)** — 새신자 자주 묻는 질문 (목업: 폼만 노출)

### 3. 예배와 교육 (`03-worship.html`)

1. **헤로** — 예배 사진/영상 + "예배와 교육" 타이틀
2. **예배 시간표** — 주일낮·주일밤·수요·금요기도회·특별예배 (표 또는 카드)
3. **생방송 / 예배 실황** — 라이브 + 다시보기 영상 그리드
4. **특별순서** — 특송·간증 영상
5. **교육 — 주일학교** — 아이콘 + 카페 링크 + 사진
6. **교육 — 중·고·대학부** — 동일 구조
7. **교육 — 청년회** — 동일 구조
8. **푸터**

### 4. 목양과 사역 (`04-care.html`)

1. **헤로** — "함께 자라고 함께 섬깁니다" 타이틀
2. **새가족 (최상단 강조)** — 등록 안내 + 모임 안내 + 학습/세례
3. **구역모임** — 소개 + 구역공과 자료 다운로드 링크
4. **경향시니어스쿨** — 프로그램 소개 + 신청
5. **평생교육원** — 커리큘럼 + 수강신청
6. **사역** — 별들의학교·신학교·선교·새소식반·복지·문화 (6칸 그리드)
7. **푸터**

### 5. 교회 활동 (`05-activity.html`)

1. **헤로** — "이번 달 교회 일정" + 작은 캘린더 위젯
2. **교회 일정 (월/주)** — 캘린더 큰 뷰
3. **주보** — 이번 주 주보 다운로드 + 지난 주보 목록
4. **교회 소식** — 영상뉴스 / 텍스트뉴스 / 교우소식 / 교단소식 (탭 또는 그리드)
5. **경향의 일주일** — 영상 시리즈 그리드
6. **푸터**

## 공통 푸터 (5개 페이지 모두 동일)

- 좌상: 로고 + 한 줄 슬로건
- 1열: 주소 / 전화 / 이메일 / 오시는 길 링크
- 2열: 메인 메뉴 5개 링크
- 3열: 온라인 헌금 / e-행정 / 주보 링크
- 4열: SNS (유튜브·인스타·페이스북)
- 하단: 저작권 + 개인정보처리방침
