# 04. 레퍼런스 — 좋은 점 / 가져올 것 / 피할 것

기획서와 사용자가 명시한 참고 사이트 3곳. 각 사이트에서 무엇을 가져올지 명시.

## 1. Grace Community Church — https://www.gracechurch.org/

**가져올 것:**
- 헤로 영상의 자연 활용 (인위적이지 않은 교회/자연 영상)
- 따뜻한 사진 톤 (실내 자연광)
- 예배 시간 정보를 헤로 바로 아래 큰 카드로
- 설교 다시보기 카드 디자인 (가로 와이드 썸네일 + 제목·본문·날짜)
- "Welcome" 페이지의 새신자 친화 톤

**피할 것:**
- 영문 특유의 큰 sans-serif는 한글에서 그대로 적용 시 인상 약함 → Pretendard로 보정

## 2. Ligonier Ministries — https://www.ligonier.org/

**가져올 것:**
- 콘텐츠 카드의 일관된 그리드 (영상·아티클·도서)
- 정보 위계의 명확함 (큰 헤딩 → 중간 라벨 → 본문 3단계)
- 검색·필터 UX는 이번 시안 범위에서 제외 (오버스펙)

**피할 것:**
- 짙은 네이비·세리프 헤딩은 우리 팔레트(Warm Earth)와 다른 톤이라 차용 안 함

**피할 것 (추가):**
- 콘텐츠 밀도가 매우 높음 → 새신자에겐 과함. 우리는 더 비워야 함.

## 3. 사랑의교회 — https://www.sarang.org/

**가져올 것:**
- 메인의 큰 영상 + 빠른 진입 카드 패턴
- 메뉴 드롭다운 정리 (한국 교회 도메인에 익숙한 메뉴 위치 감각)
- 모바일 풀스크린 메뉴 동작

**피할 것:**
- 카테고리 수가 많음 (우리는 4+1로 단순화 — 기획서 핵심 의도)
- 다중 슬라이드 캐러셀 자동재생 (접근성·로딩 문제)
- 한 화면에 정보 너무 많이 박지 않기 — 새신자가 첫 진입에서 압도되면 실패

## 4. 현재 경향교회 사이트 — https://www.ghpc.or.kr/

**역할:** 브랜드 컬러/로고 확인용 참고. 디자인은 따라가지 않음 (개편 대상).

> 시안 만들기 전 사용자가 이 사이트의 로고 색·메인 컬러를 알려주면 디자인시스템의 컬러 후보를 그쪽으로 보정 가능.

## 5. 분위기 키워드 — LOCKED

**잠긴 4개 형용사 (2026-05-18):**

1. **따뜻한** — 색온도 따뜻 (Warm Earth 팔레트), 차가운 푸른 톤 회피
