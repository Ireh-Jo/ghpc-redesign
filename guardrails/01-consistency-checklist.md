# 01. 일관성 체크리스트

5개 시안을 만든 뒤 (또는 새 시안을 추가한 뒤) 교차검증용.  
**한 항목이라도 X면 시안 폐기 또는 재작업.**

## 전역 일관성

- [ ] 5개 페이지 헤더 마크업이 동일 (현재 페이지 active 표시만 다름)
- [ ] 5개 페이지 푸터 마크업이 동일
- [ ] 컨테이너 max-width 동일 (1200px)
- [ ] 좌우 패딩 동일 (`px-5 md:px-8`)
- [ ] 섹션 사이 세로 간격 동일 (`py-20 md:py-28`)
- [ ] Tailwind CDN 버전 동일
- [ ] Pretendard CDN 동일

## 컬러

- [ ] 디자인시스템에 잠긴 팔레트(A/B/C 중 하나)의 토큰 외 컬러 사용 없음
- [ ] 본문 텍스트 모두 `--brand-ink`
- [ ] 보조 텍스트 모두 `--brand-ink-muted`
- [ ] 액센트 컬러 사용처가 5개 페이지에서 같은 의미 (예: 액센트는 항상 1차 CTA)
- [ ] 배경색 `--brand-bg` 통일 (다크 푸터는 별도 토큰)

## 타이포그래피

- [ ] 헤딩 사이즈가 `01-design-system.md` 스케일 표 안의 값만 사용
- [ ] h1/h2/h3의 weight 표와 일치
- [ ] 본문 line-height 모바일 1.65, 데스크탑 1.7
- [ ] 영문/숫자에 한해 다른 폰트 적용했다면 5개 페이지 모두 적용
- [ ] 세리프 폰트 사용 없음 (팔레트 A 잠금: Pretendard 단독)

## 컴포넌트

- [ ] `Card` 컴포넌트 클래스가 5개 페이지에서 동일
- [ ] `CTAButton` 1차/2차 변형이 5개 페이지에서 동일
- [ ] `SectionHeader` 정렬·크기 동일
- [ ] 이미지 라운드 동일 (`rounded-2xl` 또는 `rounded-xl`)
- [ ] 보더 색상 `--brand-line` 통일

## 접근성

- [ ] 모든 이미지에 alt
- [ ] 모든 버튼·링크에 클릭 영역 44x44px 이상
- [ ] 키보드 탭 순서 자연스러움
- [ ] 색 대비 WCAG AA 통과 (본문 4.5:1)
- [ ] focus-visible 스타일 존재

## 반응형

- [ ] 360px 가로 스크롤 없음
- [ ] 768px (태블릿) 깨짐 없음
- [ ] 1024px (데스크탑 기준) 정상
- [ ] 헤더 햄버거 메뉴 동작 (모바일)

## 콘텐츠

- [ ] placeholder 텍스트에 `[ ]` 또는 `(예시)` 표기로 실제 텍스트 아님을 명시
- [ ] 목사님·교인 얼굴 실사진 없음 (실루엣/추상만)
- [ ] 신학적 단언 문구 없음 (인사말·신학노선은 placeholder)
- [ ] 카테고리 5개 외 메뉴 추가 없음

## 새신자 진입 동선

- [ ] 메인에서 "주일예배 시간"이 3초 안에 보임
- [ ] 메인에서 "오시는 길"이 1뎁스로 진입 가능
- [ ] 모든 페이지에서 "새가족" 메뉴가 시각적으로 강조
- [ ] 첫 진입 시 압도되는 정보 밀도가 아님

## 50주년 어필

- [ ] 교회소개 헤로에 50주년 영상 또는 강한 시각 자산
- [ ] 다른 페이지에는 50주년 배지가 과하게 등장하지 않음 (한 곳 강조)
